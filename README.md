# gsChess-App
Les jeux d’échecs disponibles pour Linux étaient principalement accessibles via des interfaces web. Aucun jeu d’échecs plus avancé n’étant disponible nativement sur Linux, j’ai donc opté pour l’installation d’un jeu à vocation pédagogique.

Système d’exploitation testé : Fedora 43

Format du paquet : RPM

Fonctionnalités du jeu :
  - Joueur contre joueur
  - Joueur contre ordinateur
    - Cinq niveaux de difficulté
    - Fonction de retour arrière
